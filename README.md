# C++ Basic & Scientific Calculator

## Overview
This project implements a console-based calculator in C++. It supports:

### Basic Calculator
- Addition, Subtraction, Multiplication, Division
- Square, Square Root

### Scientific Calculator
- Sine, Cosine, Tangent
- Power, Logarithm (base 10), Exponential (e^x)

Users can switch between modes or quit anytime.
